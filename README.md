# Not A Bug Conf Website

## Setup

    yarn install

## Use the watcher for development

    gulp

## Compile the website

    gulp build

## Run

You can find the website in the `dist/` folder. To run it on the fly

    cd dist
    python -m SimpleHTTPServer 8000

Enjoy!
